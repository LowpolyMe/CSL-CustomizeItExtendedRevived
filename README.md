# Customize It! Extended (Revived)

This is a revival of the Customize It Extended mod by Celisuis, which I believe originated from Tai Benvenuti's Customize It! https://github.com/taibenvenuti/CustomizeIt. Both of the original projects are MIT-licensed; this fork of Celisuis' version continues maintenance and adds bug fixes/updates. Not affiliated with or endorsed by the original author.
